Rate categorization based on integer input

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Design a Java program that accepts a single integer rate (between 1 and 5) as input. Implement multiple methods, each handling a different rate. Based on the input rate, call the appropriate method to print a corresponding string message. Here are the rules: 1. If the rate is 1, print 'Very poor' 2. If the rate is 2, print 'Poor' 3. If the rate is 3, print 'Average' 4. If the rate is 4, print 'Good' 5. If the rate is 5, print 'Excellent' If the rate is less than 1 or greater than 5, print 'Invalid rate'
Sample Input:
1

Sample Output:
Very poor


Sample Input:
5

Sample Output:
Excellent



Memory limit: 256 MB
Time limit: 5 seconds
